    Python öğrenmek isteyenler için

    mofm

    2009-12-31 03:05

    Comments

    Python öğrenmek isteyenler için yararlı olacağı kanaatine vardığım web adresleri paylaşacam:

    1.İlk başta Türkçe kaynak olarak sürekli olarak takip ettiğim "istihza"yı herkese öneriyorum.Bir kez daha "istihza"yı hazırlayıp emek veren arkadaşıma teşekkür ediyorum.İstahza içerdiği Python 2.x,Python 3.x,Tkinter 2.x,PyGTK 2.x dersleriyle yerel kullanıcılar için çok büyük hizmet ediyor."İstihza" ders içeriklerine ulaşmak için buradan

    2.İkinci önerim "Bilgisayar bilimcisi gibi düşünmek" "Python ile öğrenme" başlığı altında verdiği örnekler,alıştırmalar,ünite sonu tekrarı ve ödevleri ile pythona ilgi duyanlar,öğrenmek isteyenler için iyi bir adres.Ulaşmak için buradan

    3.Paylaşan kodlar ile göz aşinalığı kazanmak ,kod okumak,öğrenmek,geliştirmek ve yazdıklarımızı paylaşmak için ActiveState

    4.Python yaratıcısı tarafından(Guido van Rossum)  hazırlanan "Python Library Reference" kitapçığına ulaşmak için buradan .Kitapçık ingilizce olsa da mutlaka bişeyler arayıp öğreneceksiniz.

    Aynı sitede "Python Programlama Dili" indeksine ulaşmak için buradan .Bu indeks bir "Python Sözlüğü" niteliğinde,pythonla ilgili aradığınız herşeyi bulabileceksiniz.

    Python ile ilgili "Modül" listesine ise buradan ulaşabilirsiniz.

    5.Ücretsiz Python kitabı.Dili ingilizce fakat yararlı olacaktır.İndirmek için buradan

    6.En önemlisi "Google".Google'ın "code search" araması yaparak python örnek kodlar bulabilirsiniz.(kod ararken "lang:python" yazmalısınız.)

    Eğer başka yararlı kaynaklar bulursam buradan paylayaşacam.Kolay gelsin.
